Welcome to GridLabDVisor!


GridLabDVisor is a Web based Application which receives data from a power distribution system simulation and analysis tool named GridLAB-D; and uses JavaScript Libraries and proper Visualization Techniques and Tools to present received data in such a way that patterns can be spotted quickly and easily by the user of the Application.

List Of Tables

Here is a list of all the tables containing in our Web Application. To start the navigation you can select the table you are interested in by clicking on it. You can also view the tables from the navigation bar on the top of the page, by clicking on "Select Table" selection and choosing the table that you desire.

Central Triplex Meter

This table contains information about the unique Central Triplex Meter of the System. To be more specific contains the values for Real & Reactive Power for each central node of the system per day.

Go to graph

Devices

This table contains information about all Devices belonging to a house. Each device has a Bid Price and a Bid Quantity that beats in the auction and the Consumed Load that actually consumes.

Go to graph

Energy Sources

This table contains information about Energy Sources of the System, which are divided in three categories. It contains information about Real & Reactive Power per node, as well as the Price that the Energy is selling.

Go to graph

Houses

In this table you can view the total load each house of the System consumed per day and how much of this load turns into Reactive Power. You can select one or more houses from the resulting list.

Go to graph

Market Pool

This table contains information about the Market Pool of the System. Specifically, it contains information about how the values of price and load change over time and per each day.

Go to graph

Nodes

In this table you can view how the Voltage belonging to each node and the Current is flowing among the Nodes of the System, as well as the System's behavior for each time quarter of a specific day.

Go to graph

Transformer

In this table you can view the value of the Power that flows into the transformer, the power that flows out of it and the power losses it probably has. This information is different for each phase of the system.

Go to graph

Devices Total Table

This is a table that contains information about Devices in table format. It includes exactly the same information with the “Devices” table, but in different form. We used the heatmap format for each factor of this table.

Go to graph

System Graph

In this table you can view the graphic topology of our whole System. It is a global diagram and it shows the connection among the nodes of the system, as well as all the houses containing by each node.

Go to graph
